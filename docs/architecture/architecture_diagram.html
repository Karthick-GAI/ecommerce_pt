Azure OpenAI · Embeddings API · 1536 dimensions
Converts product descriptions and natural-language search queries into dense 1536-dim vectors. Stored in PostgreSQL via the pgvector extension as the products.embedding column. Powers semantic search in Product Catalogue, RAG pipeline, and content-based recommendation. Generated once during dataset ingestion via generate_dataset.py.
